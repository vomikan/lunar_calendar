# ** Lunar Calendar Generator**

Генератор векторных календарей с фазами Луны на Python.

## ** О проекте**

Презентация верстается вручную в **PowerPoint**, а календарная сетка с фазами Луны генерируется автоматически. Это исключает ошибки в датах и позволяет создавать макеты на любой месяц и год за секунды.

**Особенности:**

* ✅ Векторный формат **SVG** (масштабируется без потери качества)

* ✅ Расчёт фаз Луны для любого дня

* ✅ Все настройки в файле config.toml (цвета, шрифты, размеры)

* ✅ Праздники подгружаются из CSV-таблицы

* ✅ Минимум зависимостей (работает на чистом Python 3.11+)

## ** Быстрый старт**

### **1. Установка**


```bash
# Создайте виртуальное окружение
python -m venv venv
venv\Scripts\activate

# Установите зависимости (для Python < 3.11)
pip install tomli
```


### **2. Запуск**


```bash
# Текущий месяц
python src/lunar_calendar.py

# Все месяцы 2026 года
python src/lunar_calendar.py 2026

# Конкретный месяц (март 2026)
python src/lunar_calendar.py 2026 3

# С сохранением в свой файл
python src/lunar_calendar.py 2026 3 output/my_calendar.svg
```

# ⚙️ Настройки стиля (config.toml)

Все визуальные параметры вынесены в конфигурационный файл `assets/config.toml`.

---

## 📁 Вывод

| Параметр | Описание | Пример |
|----------|----------|--------|
| `output.directory` | Папка для результатов | `"output"` |

---

## 📐 Размеры и отступы `[layout]`

| Параметр | Описание | Пример |
|----------|----------|--------|
| `cell_width` | Ширина ячейки дня (px) | `120` |
| `cell_height` | Высота ячейки дня (px) | `90` |
| `header_height` | Высота строки с днями недели (px) | `44` |
| `moon_radius` | Радиус иконки Луны (px) | `18` |
| `corner_radius` | Скругление углов ячеек (px) | `10` |
| `cell_padding` | Зазор между ячейками (px) | `6` |
| `faded_opacity` | Прозрачность дней соседних месяцев | `0.30` |

---

## 🎨 Цвета фонов `[colors.background]`

| Параметр | Описание | Пример |
|----------|----------|--------|
| `page` | Фон страницы | `"#F7F3EC"` |
| `cell_weekday` | Фон ячеек будней | `"#F0E2B6"` |
| `cell_weekend` | Фон ячеек выходных | `"#F2C4A0"` |
| `header_weekday` | Фон шапки будней | `"#E8D89A"` |
| `header_weekend` | Фон шапки выходных | `"#EDAA80"` |
| `cell_shadow` | Мягкая тень ячеек | `"rgba(0,0,0,0.07)"` |

---

## 🖊 Цвета текста `[colors.text]`

| Параметр | Описание | Пример |
|----------|----------|--------|
| `day_number_weekday` | Цифры дней (будни) | `"#5C3D0A"` |
| `day_number_weekend` | Цифры дней (выходные) | `"#7A2E0A"` |
| `header_weekday` | Текст шапки (будни) | `"#6B4A10"` |
| `header_weekend` | Текст шапки (выходные) | `"#8B3510"` |

---

## 🌙 Цвета иконки Луны `[colors.moon]`

| Параметр | Описание | Пример |
|----------|----------|--------|
| `dark` | Тёмная сторона Луны | `"#B8925A"` |
| `light` | Светлая сторона Луны | `"#FDF5E0"` |
| `ring` | Обводка иконки | `"#9A7040"` |

---

## 📅 Цвета событий `[colors.events]`

| Параметр | Описание | Пример |
|----------|----------|--------|
| `text` | Текст праздника | `"#6B2800"` |
| `text_faded` | Текст для дней соседних месяцев | `"#A07040"` |

---

## 📝 Шрифты

### Шапка `[fonts.header]`

| Параметр | Описание | Пример |
|----------|----------|--------|
| `size` | Размер шрифта (px) | `14` |
| `weight` | Насыщенность | `"bold"` |
| `letter_spacing` | Межбуквенный интервал | `2` |
| `family` | Семейство шрифтов | `"Georgia, serif"` |

### Цифры дней `[fonts.day_number]`

| Параметр | Описание | Пример |
|----------|----------|--------|
| `size` | Размер шрифта (px) | `24` |
| `weight` | Насыщенность | `"bold"` |

### События `[fonts.event]`

| Параметр | Описание | Пример |
|----------|----------|--------|
| `size` | Размер шрифта (px) | `13` |
| `style` | Стиль | `"italic"` |
| `line_height` | Высота строки (px) | `14` |
| `max_chars` | Макс. символов в строке | `13` |
| `max_lines` | Макс. строк события | `3` |
